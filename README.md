# Word of the Day Website

A simple static website for preschool and early elementary children. It shows a playful word of the day, a definition, pronunciation audio, and quick learning activities.

## Files
- `index.html` — website structure and layout
- `styles.css` — page styling and hero design
- `script.js` — daily word selection and dynamic data
- `word-of-the-day-kids-hero.webp` — hero scene image
- `word-of-the-day-logo.webp` — site logo

## How to use
1. Keep the image files in the same folder as `index.html`.
2. Open `index.html` in a browser to see the page.

## Customization
- Change the word list in `script.js` to add or update words.
- Update the meta tags in `index.html` if the live domain or share image changes.
