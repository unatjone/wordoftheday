# Word of the Day Website

A simple static website for preschool and early elementary children. It shows a hero page with the word of the day, a description, and links for more fun learning.

## Files
- `index.html` — website structure and layout
- `styles.css` — page styling and hero design
- `script.js` — daily word selection and dynamic data

## How to use
1. Place your boy image in the same folder as `index.html`.
2. Rename the image to `boy.jpg` or update the `src` in `index.html`.
3. Open `index.html` in a browser to see the page.

## Customization
- Change the word list in `script.js` to add or update words.
- Replace the button links if you want specific YouTube videos or child-friendly resources.
